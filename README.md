# PowerShell

## Descripcion
Este script es para personalizar la terminal, aparte de hacerla muy atractiva visualmente tambien le agrega comandos que facilitan trabajar con el backend.
### Como lo instalo?
- Copia el codigo del archivo PowerShellProfile
- Escribe en tu terminal ```notepad $PROFILE
- Borra todo y pega el codigo que copiaste

### Como lo utilizo?
Escribe 'commands' en la consola y te saldran todo tipo de comandos(hasta de linux)
